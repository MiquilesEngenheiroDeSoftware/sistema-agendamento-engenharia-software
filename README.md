# Sistema de Agendamento

Projeto de portfólio desenvolvido com foco em Engenharia de Software, aplicando conceitos de levantamento de requisitos, definição de escopo, arquitetura de sistemas e boas práticas de documentação.

## Objetivo
Demonstrar a capacidade de estruturar soluções de software desde a concepção do problema até a definição técnica, priorizando clareza, organização e redução de riscos.

## Principais tópicos abordados
- Engenharia de Requisitos
- Requisitos funcionais e não funcionais
- Regras de negócio
- Arquitetura em camadas
- Modelagem UML (conceitual)
- Documentação técnica

## Observação
Este projeto tem caráter demonstrativo, com ênfase nas etapas de engenharia, documentação técnica e tomada de decisão arquitetural, comuns em ambientes corporativos, não contemplando a implementação completa do sistema.
